// Simple Javascript Quiz

This project is a simple javascript quiz, there will some fundamental questions regarding javascript.
All the questions need to be answered. After answering all the questions, the score will be displayed.

In css file, the background, color, font-size, font-style, id's are represented.


Javascript is used to display the score of the participant. It will display the score in the page.
If some questions aren't answered, then it will show a message to answer the left question.


Javascript function (submitAnswers()) checks whether all the questions are answered.If not, it will display the message
and checks the correct answer and diplays the score in the top of the page.
